\begin{abstract}
This report presents a comprehensive study of lossless data compression techniques, culminating in the development of an efficient compression tool for the Algorithmic Information Theory (TAI) course. Our implementation combines Burrows-Wheeler Transform (BWT) preprocessing with adaptive Prediction by Partial Matching (PPM) models and multiple entropy coding techniques including Range Coding and Asymmetric Numeral Systems (rANS). Through iterative optimization across six major versions, we achieved an average compression ratio of 54.73\% on benchmark datasets, outperforming bzip2 by 0.20 percentage points and winning on 5 out of 8 test files. The final production system (v5.0) incorporates full pipeline parallelism, achieving 2-5× compression speedup while maintaining competitive ratios. This work demonstrates the effectiveness of combining classical preprocessing transforms with modern entropy coding and parallel processing techniques.
\end{abstract}
